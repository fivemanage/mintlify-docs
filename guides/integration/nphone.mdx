---
title: "nPhone"
description: "Learn how to integrate nPhone with Fivemanage."
---

To integrate nPhone with Fivemanage, follow these steps:

1. Open `nPhone/config/apiKeys_s.lua`
2. Set `Config_WEBHOOKS.MediaUploadMethod` to `fivemanage`
3. Set your **Fivemanage** API key in `Config_WEBHOOKS.APIKey`
